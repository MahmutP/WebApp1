### <a name="english"></a>About The Project

**WebApp1** is a collection of individual profile pages created by students as part of their front-end development practice. Each member has contributed their own HTML page to showcase their skills and learn the basics of web development, version control, and collaboration.

### Features

-   **Individual Portfolios**: Distinct HTML pages for each contributor.
-   **Simple & Clean Code**: focused on HTML5 structure.
-   **Collaborative Environment**: Demonstrates Git workflow and team cooperation.

### Getting Started

To view the project locally on your machine:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/WebApp1.git
    ```
2.  **Navigate to the project directory**
    ```bash
    cd WebApp1
    ```
3.  **Open any file**
    Open `index.html` (if available) or any specific user (e.g., `Pasha.html`, `Muhammed.html`) in your preferred browser.
